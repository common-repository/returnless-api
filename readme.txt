== Installation ==
1. Install the plugin through the Wordpress Plugin store.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Description ==

As a webshop you want to handle your returns smoothly. A bad return experience is bad for your business, and in addition, it provides you and your colleagues with extra work.

At Returnless, we promise Sweatless Returns. We automate and simplify the return process for webshops. With clear return data you can make targeted decisions and you also optimize your customer service department and assortment. Within 5 minutes you install our WooCommerce plugin and you can get started right away.

== Features ==

* Returns solutions, seamlessly integrated into your WooCommerce environment
* Process returns faster in our return portal
* Prevent returns with return analytics
* Generate vouchers to stimulate revenue retention
